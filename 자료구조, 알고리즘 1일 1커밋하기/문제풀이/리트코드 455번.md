# Assign Cookies

## 문제 분석

- g 배열: 아이들이 만족하는 쿠키의 사이즈
- s 배열: 실제 주어진 쿠키의 수
- 한 아이당 쿠키 하나 이상은 줘야 한다
- 만족하는 쿠키의 사이즈보다 큰 쿠키를 줘도 만족한다
- 크기제한  
  1 <= g.length <= 3 * 104  
  0 <= s.length <= 3 * 104  
  1 <= g[i], s[j] <= 2^31 - 1 -> 쿠키의 개수를 잴 때 `cnt[사이즈]`형태는 용량 초과가 날 수 있겠다고 생각했다.

## 구현

### CountCookie 

- 쿠키를 원하는 사람, 쿠키의 재고를 관리하는 클래스
- 쿠키를 줄 때마다 만족한 아이들의 수를 content에 저장하고 cookie에서 개수를 빼줌
- cookie의 value가 양수 -> 쿠키의 재고가 남아있음
- cookie의 value가 음수 -> 쿠키를 원하는 아이의 수보다 쿠키의 수가 적음
- 만족하는 쿠키의 사이즈보다 큰 쿠키를 줘도 만족하기 때문에 쿠키의 크기별로 정렬한 뒤,
  더 작은 쿠키를 원하던 아이들에게 큰 쿠키를 준 경우를 갱신해주었다
```java


    private class CountCookie {
        HashMap<Integer, Integer> cookie=null;
        int content;

        public CountCookie(){
            cookie=new HashMap<>();
            content=0;
        }

        public void add(int size){
            if(cookie.get(size)==null){
                cookie.put(size, 1);
            }
            else {
                cookie.put(size, cookie.get(size)+1);
            }
        }

        public void give(int size){
            if(cookie.get(size)==null){
                cookie.put(size,-1);
                return;
            }
            
            if(cookie.get(size)>0){
                content++;
            }
            
            cookie.put(size, cookie.get(size)-1);            
        }
        
        /**
        hash->keySet끼리 정렬(오름차순)->값비교
        hash.get(key)로 value비교
        음수이면->다음 hash.get(key)에 더하기 위해 uchild에 저장
        양수이면->uchild에더해줌
         */
        public int getContent(){
            List<Integer> list=new ArrayList<>(cookie.keySet());
            Collections.sort(list);
            int uchild=0;
            
            for(int li: list){
                if(cookie.get(li)<0){
                    uchild+=cookie.get(li);
                }
                else {
                    int val=cookie.get(li)+uchild;
                    if(val>0){
                        content+=Math.abs(uchild);
                        uchild=0;
                    }
                    else {
                        content+=cookie.get(li);
                        uchild=val;
                    }
                }
            }
            return content;
        }
    }

```

### findContentChildren()

```java

  public int findContentChildren(int[] g, int[] s) {

        CountCookie cookie=new CountCookie();

        for(int i=0;i<s.length;i++){
            cookie.add(s[i]);
        }

        for(int i=0;i<g.length;i++){
            cookie.give(g[i]);
        }

        return cookie.getContent();
        
    }

```
