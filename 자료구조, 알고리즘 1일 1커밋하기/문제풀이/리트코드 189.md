# Rotate Array

## 문제 분석

1. k번만큼 배열 회전  
   move() : 1번 회전하는 함수  
   k번 반복  
   움직일 때마다 (% 배열의 사이즈) -> 배열 밖으로 벗어나지 않도록 인덱스 조정

2. 인덱스 계산  
   k번 돌리면 뒤에서 k개의 값이 앞으로 옴
   인덱스 i의 위치를 (nums.length-k+i)%k 로 변경해주기

## 구현

- k가 배열의 길이보다 더 클 수 있기 때문에 나머지값으로 조정

```java

   public void rotate(int[] nums, int k) {
        int[] arr=Arrays.copyOf(nums, nums.length);
        k%=nums.length;
        k=nums.length-k;

        for(int i=0;i<nums.length;i++){
            int idx=(k+i)%nums.length;
            nums[i]=arr[idx];
        }
    }

```
