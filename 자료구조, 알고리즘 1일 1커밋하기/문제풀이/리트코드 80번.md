# Remove Duplicates from Sorted Array II

## 문제 분석
- nums: 오름차순
- 중복된 요소가 최대 2번만 나오도록 중복 제거
- 상대적인 위치는 그대로
- k: 중복 없앤 길이
- 다른 배열에 추가 공간 할당하지 말기
- O(1) 요구

## 구현
### 풀이
1. cnt로 횟수, before로 직전에 정렬한 값 체크 (초기화: cnt=0, before=-101)
2. before<nums[i] 이면 before=nums[i], nums[idx++]=before, cnt=0, k++
3. before==nums[i] && cnt<2 이면 nums[idx++]=before, cnt++, k++
4. before==nums[i] 이면 cnt++

### 테스트케이스
nums=[0,0,1,1,2,3,2,3,3]  
i=0,1,2,3,4,5,6,7  
cnt=0,1,2,1,2,3,4,1,1  
before=-101,0,1,2,3  
idx=0,1,2,3,4,5  
k=0,1,2,3,4,5  

```java

    public int removeDuplicates(int[] nums) {
        
        int cnt=0, before=Integer.MIN_VALUE, idx=0, k=0;

        for(int i=0; i<nums.length; i++){
            if(before<nums[i]){
                before=nums[i];
                nums[idx++]=before;
                cnt=1;                
                k++;
            }
            else if(before==nums[i] && cnt<2){ 
                nums[idx++]=before;
                cnt++;
                k++;
            }
            else {
                cnt++;
            }
        }

        return k;
    }

```
