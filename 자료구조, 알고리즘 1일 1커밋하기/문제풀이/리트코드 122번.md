# Best Time to Buy and Sell Stock II

## 문제 분석

- 같은 날에 주식 팔고 살 수 있음. 한번에 최대 하나이기만 하면 됨
- 최대 이익 구하기
- i번째 파는 경우 얻는 이익: 주식 판 날 이후에 사고 파는 경우까지 포함
- prices 배열 길이 1 ~ 30000
- 주식값 0 ~ 10000

## 구현

### 계획
- 주식을 산 가격보다 크기만 하면 바로 팔고 판 날의 가격을 min에 넣어주기
- 판 날의 가격이 다음 값들보다 작은 경우: [1,5,3000] 식으로 1일차에 사서 마지막날에 파는 게 가장 이득인 경우 (5-1)+(3000-5)로 계산되므로 5가 주식을 팔기 최적의 날이 아니어도 덮어씌워짐
- 판 날의 가격이 다음 값들보다 큰 경우: 주식을 팔고 나서 전날까지의 주식값은 의미가 없으므로 가장 작은 값(=주식을 살 가격)을 구하기 위한 비교대상값이 들어가야 한다.

### 코드

```java

    public int maxProfit(int[] prices) {

        int buy=Integer.MAX_VALUE;
        int sell=Integer.MIN_VALUE;
        int profit=0;
        
        for(int i=0;i<prices.length;i++){

            if(buy<prices[i]){
                profit+=(prices[i]-buy);
            }           

            buy=prices[i];
        }

        return profit;
    }

```
