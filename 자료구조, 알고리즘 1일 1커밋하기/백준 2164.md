# 카드2

## 문제분석

- 입력: 1 ≤ N ≤ 500,000
- 시간제한: 2초
- 문제 요구사항대로 아래의 과정을 모두 구현한다고 해도 O(N)이기 때문에 통과 가능하다.
    1. 1~N까지 모두 큐에 넣고
    2. 큐에서 원소 하나 제거
    3. 큐에서 원소를 하나 빼기
    4. 큐에 넣기

## 구현

```java

import java.util.LinkedList;
import java.util.Queue;
import java.util.Scanner;

public class Baek2164 {
    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        Queue<Integer> queue = new LinkedList<>();

        for (int i = 1; i <= N; i++) {
            queue.add(i);
        }

        while (queue.size() != 1) {
            queue.poll();
            int top = queue.poll();
            queue.add(top);
        }

        System.out.println(queue.poll());
    }
}


```
