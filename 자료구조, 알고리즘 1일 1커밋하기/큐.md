# 큐

먼저 들어온 원소가 먼저 나가는 선입선출(FIFO)의 자료구조

## 구현

```java

public interface QueueInterface<E> {
    public void enqueue(E x);

    public E dequeue();

    public E front();

    public boolean isEmpty();

    public void dequeueAll();
}

```

### 배열로 구현

- 배열에 낭비되는 공간이 없도록 하기 위해 큐에 값을 넣을 때 `(front+1)%queue.length` 처리

```java

public class ArrayQueue<E> implements QueueInterface<E> {

    private E[] queue;
    private int front, tail, numItems;
    private static final int DEFAULT_CAPACITY = 64;
    private final E ERROR = null;

    public ArrayQueue() {
        queue = (E[]) new Object[DEFAULT_CAPACITY];
        front = 0;
        tail = DEFAULT_CAPACITY - 1;
        numItems = 0;
    }

    public ArrayQueue(int N) {
        queue = (E[]) new Object[N];
        front = 0;
        tail = N - 1;
        numItems = 0;
    }

    @Override
    public void enqueue(E x) {
        if (isFull()) {
            System.out.println("Queue Full!!!!");
        } else {
            tail = (tail + 1) % queue.length;
            queue[tail] = x;
            ++numItems;
        }
    }

    private boolean isFull() {
        return (numItems == queue.length);
    }

    @Override
    public E dequeue() {
        if (isEmpty()) {
            System.out.println("Queue Empty!!!!");
            return ERROR;
        } else {
            E item = queue[front];
            front = (front + 1) % queue.length;
            --numItems;
            return item;
        }
    }

    @Override
    public E front() {
        if (isEmpty())
            return ERROR;
        else
            return queue[front];
    }

    @Override
    public boolean isEmpty() {
        return (numItems == 0);
    }

    @Override
    public void dequeueAll() {
        queue = (E[]) new Object[queue.length];
        front = 0;
        tail = queue.length - 1;
        numItems = 0;
    }

}

```

### 검증

```java

public class App {
    public static void main(String[] args) throws Exception {

        System.out.println("===배열로 구현한 큐===");
        QueueInterface<Integer> queue = new ArrayQueue<>();

        // 큐가 비었는지 확인
        System.out.println(queue.isEmpty());

        queue.enqueue(100);
        queue.enqueue(200);
        queue.enqueue(300);
        System.out.println("front: " + queue.front());
        System.out.println("dequeue: " + queue.dequeue());
        System.out.println("front: " + queue.front());

    }
}

```
