# 스택

후입선출(LIFO) 자료구조, 뒤에 들어온 것이 먼저 나간다.  

- 특징: 맨 위의 원소만 접근 가능하다.
- 사용예시: 자바 메모리 구조의 동적 영역에서 함수 호출할 때 사용된다.  
  자바에서는 호출된 함수가 해당함수를 호출한 함수보다 먼저 실행되어야 하기 때문이다.  
  예를 들어, A라는 함수가 B함수를 호출했을 때, B함수부터 모두 실행된 다음 A함수를 마저 실행할 수 있다.

## 구현

```java

public interface StackInterface<E> {

    public void push(E item);

    public E pop();

    public E top();

    public boolean isEmpty();

    public boolean isFull();

    public void popAll();

}

```

### 배열로 구현

- topIdx : 스택의 가장 위쪽에 있는 원소.
  - push할 때 : topIdx를 1 증가시키고 그 자리에 새 원소를 저장한다.
  - pop할 때 : topIdx가 가리키고 있는 원소를 리턴하고 topIdx를 1 감소시킨다. 

```java

public class ArrayStack<E> implements StackInterface {

    private E[] stack;
    private int topIdx;
    private static final int DEFAULT_CAPACITY = 60;
    private final E ERR = null;

    public ArrayStack() {
        stack = (E[]) new Object[DEFAULT_CAPACITY];
        topIdx = -1;
    }

    public ArrayStack(int n) {
        stack = (E[]) new Object[n];
        topIdx = -1;
    }

    @Override
    public void push(Object item) {
        if (isFull()) {
            System.out.println("stack is full");
        } else {
            stack[++topIdx] = (E) item;
        }
    }

    @Override
    public Object pop() {
        if (isEmpty()) {
            return ERR;
        } else {
            return stack[topIdx--];
        }
    }

    @Override
    public Object top() {
        if (isEmpty()) {
            return ERR;
        } else {
            return stack[topIdx];
        }
    }

    @Override
    public boolean isEmpty() {
        return (topIdx == -1);
    }

    @Override
    public void popAll() {
        stack = (E[]) new Object[stack.length];
        topIdx = -1;
    }

    @Override
    public boolean isFull() {
        return stack.length - 1 <= topIdx;
    }

}


```
