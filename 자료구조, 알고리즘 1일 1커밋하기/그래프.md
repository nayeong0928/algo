# 그래프

현상이나 사물을 정점과 간선으로 표현한 것
- 정점: 대상이나 개체
- 간선: 관계

## 너비 우선 탐색 (BFS)

1. 루트의 자식을 차례로 방문
2. 방문한 정점의 자식을 차례로 방문
3. 더 이상 방문할 곳이 없을 때까지 반복

수행시간: O(V+E) -> 시작 정점에서 방문할 수 없는 정점의 경우 추가 BFS가 필요하기 때문.

## 깊이 우선 탐색 (DFS)

1. 루트의 자식 중 하나를 골라 방문
2. 방문한 정점의 자식을 하나 골라 방문
3. 더 이상 방문할 곳이 없을 때까지 방문
4. 아직 방문하지 않은 정점을 방문, 2->3을 반복
5. 시작 정점으로부터 방문할 수 있는 모든 정점을 방문할 때까지 반복

수행시간: O(V+E) -> 시작 정점에서 방문할 수 없는 정점의 경우 추가 DFS가 필요하기 때문.
