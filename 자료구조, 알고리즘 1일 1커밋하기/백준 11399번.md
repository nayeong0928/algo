# ATM - 삽입정렬

## 문제 분석

필요한 시간의 합의 최소값을 구하려면 ATM에서 돈을 뽑는 시간이 오름차순이 되는 방식으로 줄을 서야 한다.  
때문에 오름차순 정렬 후 기다리는 시간의 합을 구하는 방식으로 문제를 풀었다.  
  
입력 크기 N이 1000 이하이고 시간제한이 1초이기 때문에 최악 시간복잡도 n^2이 나오더라도 통과할 수 있다.  
버블/병합/삽입/선택/퀵정렬 모두 사용할 수 있지만 그 중 삽입 정렬을 이용하여 풀어보았다.  

## 구현

```java

import java.util.Scanner;

public class sol11399 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        int N = sc.nextInt();
        int[] arr = new int[N];

        for (int n = 0; n < N; n++) {
            arr[n] = sc.nextInt();
        }

        sc.close();

        selectionSort(arr);

        for (int i = 1; i < arr.length; i++) {
            arr[i] += arr[i - 1];
        }

        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            sum += arr[i];
        }

        System.out.println(sum);
    }

    private static void selectionSort(int[] arr) {

        for (int last = 1; last < arr.length; last++) {

            int i = last - 1;

            while (i >= 0 && arr[i] >= arr[i + 1]) {
                int tmp = arr[i + 1];
                arr[i + 1] = arr[i];
                arr[i] = tmp;
                i--;
            }
        }
    }

}


```
