# 버블 정렬

나란히 있는 두 원소를 비교하여 더 큰 원소를 오른쪽으로 보내서 정렬하는 방법

- 최악 시간복잡도: O(n^2)
- 최선 시간복잡도: O(n^2)

## 구현

```java


    private static void bubbleSort(int[] arr) {

        for (int last = arr.length - 1; last > 0; last--) {
            for (int i = 0; i < last; i++) {
                if (arr[i] > arr[i + 1]) {
                    int tmp = arr[i];
                    arr[i] = arr[i + 1];
                    arr[i + 1] = tmp;
                }
                count++;
            }
        }

    }

```
