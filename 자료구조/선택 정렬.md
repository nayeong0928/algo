# 선택정렬

배열을 순회하며 가장 큰 값을 찾아 배열의 맨 끝자리에 두는 과정을 반복하는 정렬 방법.
배열이 정렬되어 있든, 정렬되어 있지 않든 배열을 순회하기 때문에 최악/평균/최선 시간복잡도가 같다.

  - 최악 시간복잡도: O(n^2)
  - 평균 시간복잡도: O(n^2)
  - 최선 시간복잡도: O(n^2)
  
## 구현

- end: 정렬되지 않은 배열의 맨 끝자리 인덱스
- max: 가장 큰 값의 인덱스

```java

    private static void selectionSort(int[] arr) {

        int end = arr.length - 1;

        while (end > 0) {

            int max = end;

            for (int i = 0; i < end; i++) {

                if (arr[i] > arr[max]) {
                    max = i;
                }
            }

            int tmp = arr[max];
            arr[max] = arr[end];
            arr[end] = tmp;
            end--;
        }

    }

```
