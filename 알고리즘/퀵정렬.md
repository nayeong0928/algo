# 퀵정렬

기준이 되는 피봇을 정해서 피봇보다 작은 값은 피봇의 앞에, 피봇보다 큰 값은 피봇의 뒤에 넣는 정렬 방식

- 평균 시간 복잡도: O(n log n)
- 최악 시간 복잡도: O(n^2)
- 최고 시간 복잡도: O(log n)  
  
  
## 구현
피봇을 start와 end의 중간으로 골랐다.  
low: 배열의 start에서부터 end 방향으로 피봇보다 작은 값인지 체크, 피봇보다 큰 값이 나오면 위치를 바꿔야 하니까 멈춘다.  
high: 배열의 end부터 start 방향으로 피봇보다 큰 값인지 체크, 피봇보다 작은 값이 나오면 위치를 바꿔야 하니까 멈춘다.   


```java

    private static void quickSort(int[] arr, int start, int end) {
        if (start >= end) {
            return;
        }

        int middle = start + (end - start) / 2;
        int pivot = arr[middle];
        int low = start;
        int high = end;

        while (low <= high) {

            while (arr[low] <= pivot) {
                low++;
            }

            while (arr[high] > pivot) {
                high--;
            }

            if (low < high) {
                int temp = arr[low];
                arr[low] = arr[high];
                arr[high] = temp;
                low++;
                high++;
            }

        }

        if (start < high) {
            quickSort(arr, start, high);
        }

        if (start > low) {
            quickSort(arr, low, end);
        }

    }

```
